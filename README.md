# 回声网络

## 项目介绍

“回声网络”是一个旨在复兴深度内容创作与交流的在线多人博客与内容管理系统。

### 技术选型

前后端分离开发模式

- 前端：Vue3 + Element Plus + Axios
- 后端：JDBC + MySQL + Servlet
