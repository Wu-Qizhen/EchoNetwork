## 使用前导入

在 `main.js` 或 `main.ts` 中添加：

```javascript
import '@/aethex/foundation/colors/colors.css'
import '@/aethex/foundation/typography/font.css'
import '@/aethex/foundation/typography/layout.css'
import '@/aethex/foundation/ui/components.css'
```
